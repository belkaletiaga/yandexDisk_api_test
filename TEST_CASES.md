## Тест-кейсы API Яндекс.Диска

### TC - 01: Получение метаинформации о диске пользователя (GET /v1/disk)
#### Тег: Позитивный тест-кейс
| Field                    | Value                                                                                                                                                                                        |
|--------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Тест-кейс ID**         | TC - 01<br>                                                                                                                                                                                  |
| **Описание**             | Успешное получение информации о диске (общий объём, использованное пространство, системные папки и т.д.).<br>                                                                                |                                                                                                                                                                                                                                                                                                                                                                                                                                                                 
| **Предусловие:**         | 1. Сервис запущен.<br> 2. У пользователя есть валидный OAuth-токен.<br>                                                                                                                      |
| **Шаги**                 | 1. Сформировать GET запрос на /v1/disk/  с заголовком "Authorization: OAuth <token>".<br>                                                                                                    |  
| **Ожидаемый результат:** | 1. HTTP статус ответа = 200.<br>2. В теле ответа присутствуют поля: total_space, used_space, trash_size, system_folders — неотрицательные числа; used_space ≤ total_space; user- не null<br> |

## TC-03: Создание папки (PUT /v1/disk/resources) 
#### Тег: Позитивный тест-кейс
| Field                    | Value                                                                                                                                                                          |
|--------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Тест-кейс ID**         | TC - 03<br>                                                                                                                                                                    |
| **Описание**             | Успешное создание новой папки на диске.<br>                                                                                                                                    |                                                                                                                                                                                                                                                                                                                                                                                                                                                                 
| **Предусловие:**         | 1. Сервис запущен.<br> 2. У пользователя есть валидный OAuth-токен.<br> 3. Папки с указанным путём на Диске не существует.<br>                                                 |
| **Шаги**                 | 1. Сформировать и отправить PUT запрос на /v1/disk/resources с параметром path=/new_folder. с заголовком "Authorization: OAuth <token>."<br>                                   |  
| **Ожидаемый результат:** | 1. HTTP статус ответа = 201.<br>2. В теле ответа содержится объект Link с method="GET" и href, ведущим на метаинформацию о папке. <br>3. По пути /new_folder появляется папка. |

## TC-04: Получение метаинформации созданной папки  (GET /v1/disk/resources)
#### Тег: Позитивный тест-кейс
| Field                    | Value                                                                                                                                            |
|--------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| **Тест-кейс ID**         | TC - 04<br>                                                                                                                                      |
| **Описание**             | Успешное получение метаинформации о существующей папке.<br>                                                                                      |                                                                                                                                                                                                                                                                                                                                                                                                                                                                 
| **Предусловие:**         | 1. Сервис запущен.<br> 2. У пользователя есть валидный OAuth-токен.<br> 3. На диске существует папка по пути /source_folder.<br>                 |
| **Шаги**                 | 1. Сформировать и отправить GET запрос на /v1/disk/resources с параметром path=/new_folder. с заголовком "Authorization: OAuth <token>."<br>     |  
| **Ожидаемый результат:** | 1. HTTP статус ответа = 200.<br>2. В теле ответа присутствуют поля: path, type, name, created, modified. <br>3. Значение поля type равно "dir".  |

## TC-08: Удаление папки (DELETE /v1/disk/resources)
#### Тег: Позитивный тест-кейс
| Field                    | Value                                                                                                                    |
|--------------------------|--------------------------------------------------------------------------------------------------------------------------|
| **Тест-кейс ID**         | TC-08<br>                                                                                                                |
| **Описание**             | Безвозвратное удаление папки(не помещая в корзину).<br>                                                                  |
